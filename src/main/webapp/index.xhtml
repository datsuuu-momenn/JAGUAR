オートメーションプラクティス
Text input Field:
・当用户在输入框中输入文字时，会触发oninput事件并调用script.js中的handleInputChange()函数
Textarea:
・可通过拖动右下角调整文本区域大小
Pre-Filled Text Field:
・可通过value属性预填充文本内容
button ・可通过onclick事件，触发script.js中的buttonFuction1()函数
placeholder Text Field:
・可通过placeholder属性，在文本框内容为空时显示提示信息
Read-Only Text Field:
・可通过readonly属性，使文本框内容无法被修改
HTML SVG with rect:
・SVG矩形元素支持多种动画触发方式：
1. mouseover: 鼠标悬停时改变填充颜色
2. mouseout: 鼠标移出恢复原色
3. click: （初次）点击时增加边框宽度
4. JavaScript控制: （点击触发）可通过handleSVGClick()函数改变颜色

This Text is Green

・p标签:添加文本内容
・style属性:改变文本颜色、字体大小等
Input Slider Control: ・oninput:拖动滑块时实时触发sliderInputFuction()
・onclick:点击触发sliderClickFuction()
・onchange:滑块值发生变化且用户完成交互时触发sliderChangeFuction()
・onmousemove:鼠标在滑块上移动时实时触发sliderMouseMoveFuction()实时显示当前值的浮动提示框
・onmouseover:悬停触发sliderMouseOverFuction()增加滑块的不透明度
・onmouseout:移出触发sliderMouseOutFuction()移除值显示元素并降低滑块不透明度
・onfocus:点击聚焦圆点触发sliderFocusFuction()圆点变为深绿色
・onblur:点击失焦圆点触发sliderBlurFuction()圆点恢复默认颜色
・progress标签:进度条
・value属性:设置进度条的当前值为50%
・max属性:设置进度条的最大值为100
・style属性:设置进度条的宽度
 		Select Dropdown ・当前Select的功能：选择不同选项来更新进度条的值

・meter标签：进度条

・style属性：设置进度条的宽度
